# Code of Conduct

## Our pledge

We want CodePet to be a welcoming, harassment-free project for everyone,
regardless of experience level, background, or identity.

## Our standards

Examples of behavior that helps:

- Being kind, respectful, and constructive.
- Welcoming newcomers and helping them get started.
- Accepting feedback gracefully and focusing on what's best for the project.

Examples of unacceptable behavior:

- Harassment, insults, or derogatory comments.
- Personal or political attacks.
- Publishing others' private information without permission.

## Enforcement

Project maintainers are responsible for clarifying standards and may remove
comments, commits, code, issues, and other contributions that violate this
Code of Conduct, or ban temporarily or permanently any contributor for
behaviors they deem inappropriate.

Report issues by opening a confidential issue or contacting the maintainer
via their GitHub profile.

This Code of Conduct is adapted from the
[Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
